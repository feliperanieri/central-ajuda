Central de Ajuda

Dúvidas frequentes

  1. 1. Qual a diferença entre WorkFlow e FollowUp?
    1. FollowUp: Significa buscar uma resposta para algo que foi requisitado, solicitado. Quando se faz o "follow up" você está buscando um retorno do que foi questionado e que ainda não obteve resposta. O processo Follow Up permite definir uma data retorno, onde o sistema irá gerar uma pendencia para você no dia informado. WorkFlow: É a modelização e a gestão do conjunto das tarefas à realizar, de acordo com um conjunto de regras definidas e seus respectivos atores. O processo Work Flow possibilita a criação de uma sequencia de tarefas a serem executadas por seus executores.
  2. 2. Onde é definido qual tipo de cliente faz parte do processo?
    1. Através do parâmetro "Controla Clientes" é possível definir qual tipo de cliente faz parte do processo. Para isto realize o seguinte procedimento: - Menu > Cadastro > Processos > Mapear Processos - Selecione o Processo - Clique no ícone "Formulário" - Habilite o parâmetro "Controla Clientes" Obs: Ao selecionar o parâmetro citado acima sera aberto um novo campo para selecionar o tipo de cliente.
  3. 3. É possível alterar um formulário já preenchido?
    1. Sim, porém somente o Gestor ou Usuário Responsável poderão alterar os formulários já preenchidos. Para isso realize o seguinte procedimento: - Menu > Cadastro > Processos > Mapear Processos - Selecione o Processo - Clique no ícone "Formulário" - Habilite o parâmetro: Permite Gestor Alterar Formulário (Gestor do Processo / Gestor da Divisão / Usuário Responsável)
  4. 4. O que são regras para aprovação?
    1. Regra de aprovação é uma condição que criamos para permitir que o sistema defina qual sera a próxima tarefa a ser executada baseada no formulário. IMPORTANTE: As regras podem ser configuradas por Tarefa ou por Executor, ou seja, podemos criar uma condição para definir qual a próxima tarefa e/ou qual executor irá realiza-la.
